# smux (switch between tmux sessions and windows with keywords)

## Install 

```sh
git clone git@github.com:mainliufeng/smux.git;
cp ./smux/smux /usr/local/bin/;
sh ./smux/smux_plugin.tmux;
```

## Usage 

prefix-s in tmux
